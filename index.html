Simplifying Expressions

  1. O(n + 10) => O(n) // Drop the constant term.
  2. O(100 * n) => O(n) // Drop the constant factor.
  3. O(25) => O(1) // Constant time complexity.
  4. O(n^2 + n^3) => O(n^3) // Dominant term.
  5. O(n + n + n + n) => O(n) // Combine like terms.
  6. O(1000 * log(n) + n) => O(n) // Dominant term.
  7. O(1000 * n * log(n) + n) => O(n * log(n)) // Dominant term.
  8. O(2^n + n^2) => O(2^n) // Dominant term.
  9. O(5 + 3 + 1) => O(1) // Constant time complexity.
  10. O(n + n^(1/2) + n^2 + n * log(n)^10) => O(n^2) // Dominant term.
